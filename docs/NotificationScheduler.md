# NotificationScheduler - Планировщик уведомлений и опросников

## Обзор

`NotificationScheduler` - это класс для планирования и автоматической отправки уведомлений и опросников пользователям бота. Он использует библиотеку `node-cron` для планирования задач и интегрируется с `SurveyService` для отправки опросников.

## Возможности

### 📅 Запланированные задачи

1. **Утренние опросники** - каждый день в 9:00
2. **Вечерние опросники** - каждый день в 20:00  
3. **Еженедельные опросники** - по воскресеньям в 18:00
4. **Еженедельные уведомления** - по понедельникам и четвергам в 10:00
5. **Ежедневная проверка здоровья** - каждый день в 6:00
6. **Ежемесячная статистика** - первое число месяца в 9:00

### 📊 Отчеты администраторам

Система автоматически отправляет отчеты администраторам после выполнения каждой задачи:
- Статистика успешных/неуспешных отправок
- Процент успеха
- Детали ошибок (первые 10)
- Временные метки выполнения

## Использование

### Инициализация

```javascript
const NotificationScheduler = require('./src/jobs/NotificationScheduler');
const SubscriptionService = require('./src/services/SubscriptionService');
const AdminService = require('./src/services/AdminService');

const scheduler = new NotificationScheduler(subscriptionService, adminService);
```

### Запуск планировщика

```javascript
// Запускает все запланированные задачи
scheduler.start();

// Проверка статуса
const status = scheduler.getStatus();
console.log(status);
```

### Остановка планировщика

```javascript
// Останавливает все активные задачи
scheduler.stop();
```

### Ручной запуск опросников (для тестирования)

```javascript
// Запуск утренних опросников
await scheduler.runMorningSurveysNow();

// Запуск вечерних опросников
await scheduler.runEveningSurveysNow();

// Запуск еженедельных опросников
await scheduler.runWeeklySurveysNow();

// Запуск всех опросников
await scheduler.runAllSurveysNow();
```

## Типы опросников

### 🌅 Утренний опросник (morning_mood)

**Время:** 9:00 каждый день

**Вопросы:**
1. **Уровень энергии:**
   - 😴 Сонный → Розмарин, Лимон, Мята
   - 😊 Бодрый → Апельсин, Грейпфрут, Бергамот
   - 😌 Спокойный → Лаванда, Ромашка, Иланг-иланг
   - 😤 Напряженный → Лаванда, Ветивер, Бергамот

2. **Планы на день:**
   - 💼 Работа/учеба → Розмарин, Лимон, Базилик
   - 🏃‍♀️ Спорт/активность → Эвкалипт, Мята, Пихта
   - 😴 Отдых/релакс → Лаванда, Ромашка, Сандал
   - 🎨 Творчество → Бергамот, Иланг-иланг, Пачули

### 🌙 Вечерний опросник (evening_mood)

**Время:** 20:00 каждый день

**Вопросы:**
1. **Оценка дня:**
   - 😊 Отлично! → Лаванда, Ромашка, Сандал
   - 😐 Нормально → Бергамот, Иланг-иланг, Лаванда
   - 😔 Устал → Лаванда, Ветивер, Ромашка
   - 😤 Стресс → Лаванда, Бергамот, Иланг-иланг

2. **Планы на вечер:**
   - 😴 Ранний сон → Лаванда, Ромашка, Валериана
   - 📚 Чтение/учеба → Розмарин, Лимон, Базилик
   - 🎵 Музыка/медитация → Сандал, Ладан, Иланг-иланг
   - 💆‍♀️ Массаж/ванна → Лаванда, Ромашка, Бергамот

### 📅 Еженедельный опросник (weekly_review)

**Время:** 18:00 по воскресеньям

**Вопросы:**
1. **Использованные масла за неделю:**
   - Лаванда, Мята, Лимон, Розмарин, Другие, Не использовал

2. **Замеченный эффект:**
   - 😴 Лучше сплю
   - ⚡ Больше энергии
   - 😌 Меньше стресса
   - 🤔 Пока не заметил

## Обработка ответов

Каждый ответ на опросник генерирует персонализированную рекомендацию с:
- Подходящими эфирными маслами
- Конкретными советами по применению
- Дозировками и способами использования

## Логирование

Система ведет подробные логи:
- Запуск/остановка планировщика
- Выполнение каждой задачи
- Успешные/неуспешные отправки
- Ошибки с деталями
- Отчеты администраторам

## Тестирование

Для тестирования используйте скрипт:

```bash
node scripts/testSurveys.js
```

Этот скрипт:
- Создает мок-сервисы
- Запускает планировщик
- Тестирует ручной запуск всех опросников
- Показывает статус и отчеты

## Зависимости

- `node-cron` - планирование задач
- `SurveyService` - отправка опросников
- `SubscriptionService` - управление подписчиками
- `AdminService` - отправка отчетов администраторам
- `logger` - логирование

## Конфигурация

Все временные настройки используют часовой пояс `Europe/Moscow`.

Для изменения расписания отредактируйте соответствующие методы:
- `scheduleMorningSurveys()`
- `scheduleEveningSurveys()`
- `scheduleWeeklySurveys()`
- `scheduleWeeklyNotifications()`
- `scheduleDailyHealthCheck()`
- `scheduleMonthlyStats()`
