## 🎯 Как это работает

### 🌅 Утренние рекомендации
1. **Пользователь подписывается** на утренние рекомендации
2. **Указывает время** (например, 08:30)
3. **Система генерирует** персонализированную рекомендацию
4. **В указанное время** отправляется сообщение с маслами и музыкой
5. **Рекомендация сохраняется** в базе данных

### 🌆 Вечерние рекомендации
1. **Пользователь подписывается** на вечерние рекомендации
2. **Указывает время** (например, 21:00)
3. **Система создает** расслабляющую рекомендацию
4. **В указанное время** отправляется сообщение для вечернего отдыха
5. **Рекомендация адаптируется** под настроение пользователя

## 🎨 Персонализация

### Факторы персонализации:
- **Любимые масла** пользователя из профиля
- **История использования** и предпочтения
- **Текущее настроение** и цели
- **Время суток** (утренние vs вечерние масла)

### Алгоритм выбора:
```javascript
// Утренние масла: энергичные и бодрящие

// Вечерние масла: расслабляющие и успокаивающие

// Добавляются любимые масла пользователя
selected_oils = [...favorite_oils, ...time_based_oils]
```

## 💬 Интеграция с Telegram

### Команды бота:
```javascript
/subscribe_morning 08:30    // Подписка на утренние
/subscribe_evening 21:00    // Подписка на вечерние
/unsubscribe_morning        // Отписка от утренних
/unsubscribe_evening        // Отписка от вечерних
/subscription_status        // Статус подписок
```

### Инлайн-кнопки:
- 🌅 Подписаться на утренние

- ❌ Отписаться от утренних
- ❌ Отписаться от вечерних

- 📊 Статус подписок
