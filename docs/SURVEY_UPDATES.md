# Обновления NotificationScheduler для опросников

## Что было добавлено

### 1. Планирование опросников
- **Утренние опросники** - каждый день в 9:00
- **Вечерние опросники** - каждый день в 20:00
- **Еженедельные опросники** - по воскресеньям в 18:00

### 2. Новые методы планирования
```javascript
scheduleMorningSurveys()    // Планирует утренние опросники
scheduleEveningSurveys()    // Планирует вечерние опросники
scheduleWeeklySurveys()     // Планирует еженедельные опросники
```

### 3. Улучшенные методы отправки
- Детальная обработка ошибок
- Сбор статистики успешных/неуспешных отправок
- Автоматические отчеты администраторам
- Задержки между отправками для избежания rate limits

### 4. Методы для ручного запуска
```javascript
runMorningSurveysNow()      // Запуск утренних опросников
runEveningSurveysNow()      // Запуск вечерних опросников
runWeeklySurveysNow()       // Запуск еженедельных опросников
runAllSurveysNow()          // Запуск всех опросников
```

### 5. Расширенная статистика
- Отдельные отчеты для каждого типа опросников
- Процент успеха отправок
- Детали ошибок (первые 10)
- Временные метки выполнения

### 6. Улучшенный статус
```javascript
{
  isRunning: boolean,
  activeJobs: string[],
  jobCount: number,
  surveyJobs: string[],           // Только опросники
  notificationJobs: string[],     // Только уведомления
  scheduledTasks: {               // Расписание всех задач
    morningSurveys: '9:00 daily',
    eveningSurveys: '20:00 daily',
    weeklySurveys: 'Sunday 18:00',
    // ... остальные задачи
  }
}
```

## Интеграция с SurveyService

NotificationScheduler теперь полностью интегрирован с SurveyService:
- Автоматически создает экземпляр SurveyService
- Использует методы `sendMorningSurvey()`, `sendEveningSurvey()`, `sendWeeklySurvey()`
- Обрабатывает результаты отправки
- Логирует все действия

## Тестирование

Создан тестовый скрипт `scripts/testSurveys.js`:
- Создает мок-сервисы для тестирования
- Запускает планировщик
- Тестирует ручной запуск всех опросников
- Показывает статус и отчеты

## Использование

### Запуск планировщика
```javascript
const scheduler = new NotificationScheduler(subscriptionService, adminService);
scheduler.start(); // Запускает все задачи, включая опросники
```

### Проверка статуса
```javascript
const status = scheduler.getStatus();
console.log('Активные опросники:', status.surveyJobs);
console.log('Расписание:', status.scheduledTasks);
```

### Ручное тестирование
```javascript
// Тестирование утренних опросников
await scheduler.runMorningSurveysNow();
```

## Логирование

Добавлено подробное логирование:
- Запуск каждой задачи опросников
- Успешные/неуспешные отправки
- Детали ошибок
- Отчеты администраторам

## Часовой пояс

Все задачи используют часовой пояс `Europe/Moscow` для корректной работы в российском времени.

- Не изменяют существующие методы
- Добавляют новые возможности
- Сохраняют обратную совместимость
