# Система обработки грамматических ошибок

## Обзор

Система автоматического исправления грамматических ошибок в названиях эфирных масел позволяет боту корректно обрабатывать запросы пользователей даже при наличии опечаток, ошибок в написании и грамматических форм.

## Как это работает

### 1. Словарь опечаток и грамматических форм

Система использует словарь `typoDictionary` в файле `src/utils/textNormalizer.js`, который содержит:

```javascript
const typoDictionary = {
  // Лаванда и варианты с ошибками
  'лавонда': 'лаванда',
  'лавонде': 'лаванда',
  'лавандо': 'лаванда',
  'лаванде': 'лаванда',
  'лаванды': 'лаванда',
  'лавандой': 'лаванда',
  'лаванду': 'лаванда',
  'лаванд': 'лаванда',
  
  // Мята и варианты с ошибками
  'мят': 'мята',
  'мяту': 'мята',
  'мятой': 'мята',
  
  // Эвкалипт и варианты с ошибками
  'евкалипт': 'эвкалипт',
  'евкалипта': 'эвкалипт',
  'евкалипту': 'эвкалипт',
  'евкалиптом': 'эвкалипт',
  // ... и другие
};
```

### 2. Процесс обработки

1. **Получение текста** от пользователя
2. **Нормализация** - приведение к нижнему регистру и удаление лишних пробелов
3. **Исправление опечаток** - замена неправильных форм на правильные
4. **Извлечение названия масла** - поиск исправленного названия в тексте
5. **Обработка запроса** - предоставление информации о найденном масле

### 3. Интеграция с SmartRouter

SmartRouter использует систему нормализации для:

- Поиска масел в тексте запроса
- Обработки неоднозначных случаев
- Предоставления релевантных ответов

## Примеры работы

### Пример 1: Простая ошибка
```
👤 Пользователь: "лавонде"
🤖 Бот понимает: "лаванда"
📋 Ответ: Полная информация о лаванде
```

### Пример 2: Ошибка в контексте
```
👤 Пользователь: "расскажи про лавонде"
🤖 Бот понимает: "расскажи о лаванде"
📋 Ответ: Информация о лаванде
```

### Пример 3: Множественные ошибки
```
👤 Пользователь: "лавонде и мят"
🤖 Бот понимает: "лаванда и мята"
📋 Ответ: Информация о лаванде (первое найденное масло)
```

## Дополнительные возможности

### 1. Общие опечатки
Система также исправляет общие опечатки:
- `ё` → `е`
- `йо` → `ё`
- `щ` → `ш`
- `ъ` → `ь`
- `э` → `е`

### 2. Нормализация фраз
```
"расскажи про" → "расскажи о"
"как использовать" → "как применять"
"эфирное масло" → "масло"
```

## Тестирование

Для тестирования системы используются скрипты:

- `scripts/testGrammarCorrection.js` - тестирование исправления ошибок
- `scripts/testBotWithGrammarErrors.js` - тестирование работы бота с ошибками
- `scripts/testFullBotResponse.js` - демонстрация полного ответа

### Запуск тестов
```bash
node scripts/testGrammarCorrection.js
node scripts/testBotWithGrammarErrors.js
node scripts/testFullBotResponse.js
```

## Преимущества

1. **Улучшенный UX** - пользователи получают ответы даже при ошибках
2. **Снижение нагрузки** - меньше нераспознанных запросов
3. **Естественность** - бот понимает человеческие ошибки
4. **Масштабируемость** - легко добавлять новые формы слов
5. **Надежность** - система работает даже при серьезных ошибках

## Расширение системы

Для добавления новых масел или форм слов:

1. Добавить записи в `typoDictionary`
2. Добавить основное название в `mainOilNames`
3. Протестировать с помощью тестовых скриптов

## Заключение

Система обработки грамматических ошибок значительно улучшает пользовательский опыт, позволяя боту корректно понимать запросы даже при наличии опечаток и ошибок в написании названий эфирных масел.
