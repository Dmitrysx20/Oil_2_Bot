# 🔍 Perplexity Integration - Краткое резюме

## ✅ Что реализовано

### 🤖 AI-сервис с интеграцией:
- **Perplexity API** - основной источник рекомендаций
- **OpenAI API** - резервный источник
- **Локальные заглушки** - fallback при недоступности API

### 🔄 Система приоритетов:
1. **Perplexity** → умные рекомендации с поиском
2. **OpenAI** → качественные ответы
3. **Fallback** → базовые рекомендации

### 📝 Типы рекомендаций:
- **Базовые** - по ключевым словам (энергия, спокойствие, сон)
- **Медицинские** - по симптомам (головная боль, простуда, стресс)

## 🔧 Техническая реализация

### API методы:
```javascript
// Базовые рекомендации
await aiService.getBasicRecommendation({
  keywords: ['энергия', 'бодрость'],
  chatId: 123456789
})

// Медицинские рекомендации
await aiService.getMedicalRecommendation({
  medicalInfo: { category: 'головная_боль' },
  chatId: 123456789
})

// Прямой вызов Perplexity
await aiService.getPerplexityRecommendation(prompt)
```

### Конфигурация:
```javascript
perplexity: {
  apiKey: process.env.PERPLEXITY_API_KEY,
  model: 'llama-3.1-sonar-small-128k-online'
}
```

## 🎯 Как это работает

### 1. Пользователь запрашивает рекомендацию
```
"Нужна энергия"
"Болит голова"
"Хочу расслабиться"
```

### 2. SmartRouter определяет тип запроса
- Ключевые слова → базовые рекомендации
- Медицинские симптомы → медицинские рекомендации

### 3. AIService обрабатывает запрос
- Проверяет доступность Perplexity API
- Формирует промпт на русском языке
- Отправляет запрос к API

### 4. Получает и форматирует ответ
- Красивое форматирование в Markdown
- Интерактивные кнопки
- Информация об источнике

## 📊 Статистика использования

### Источники ответов:
- `source: 'perplexity'` - от Perplexity API
- `source: 'openai'` - от OpenAI API  
- `source: 'fallback'` - локальные заглушки

### Метрики:
- `tokensUsed` - использованные токены
- `type` - тип рекомендации
- `searchUsed` - использовался ли поиск
