# 🔧 Настройка нового токена бота

## ❌ Проблема: Текущий токен недействителен

Ошибка 401 "Unauthorized" означает, что токен `7721336065:AAH_0a9z8W6Lpr9EnFhroKU2ua4Df_PihBA` недействителен.

## ✅ Решение: Создание нового бота

### Шаг 1: Создание бота в Telegram

1. **Откройте Telegram**
2. **Найдите @BotFather**
3. **Отправьте команду** `/newbot`
4. **Введите имя бота:** `Aromatic Story`
5. **Введите username:** `aromatic_story_bot` (должен заканчиваться на `_bot`)
6. **Сохраните новый токен**

### Шаг 2: Проверка токена

```bash
# Замените YOUR_NEW_TOKEN на полученный токен
curl -s "https://api.telegram.org/botYOUR_NEW_TOKEN/getMe" | python3 -m json.tool
```

Должен вернуться ответ:
```json
{
    "ok": true,
    "result": {
        "id": 1234567890,
        "is_bot": true,
        "first_name": "Aromatic Story",
        "username": "aromatic_story_bot"
    }
}
```

### Шаг 3: Обновление токена в Railway

1. **Откройте Railway Dashboard**
2. **Перейдите в Variables**
3. **Найдите `TELEGRAM_BOT_TOKEN`**
4. **Нажмите на значок редактирования**
5. **Вставьте новый токен**
6. **Сохраните изменения**

### Шаг 4: Локальная настройка (для тестирования)

```bash
# Установите новый токен локально
export TELEGRAM_BOT_TOKEN="YOUR_NEW_TOKEN"

# Проверьте
echo $TELEGRAM_BOT_TOKEN

# Перезапустите бота
pkill -f "node.*app.js"
ENABLE_WEBHOOK=true npm start
```

### Шаг 5: Настройка webhook

```bash
# Установите webhook
node scripts/setWebhook.js set

# Проверьте статус
node scripts/setWebhook.js info
```

## 🔍 Альтернативные решения

### Вариант A: Использование существующего бота

Если у вас уже есть рабочий бот:
1. Найдите @BotFather
2. Отправьте `/mybots`
3. Выберите нужного бота
4. Нажмите "API Token"
5. Скопируйте токен

### Вариант B: Проверка текущего бота

```bash
# Проверьте, существует ли бот
curl -s "https://api.telegram.org/bot7721336065:AAH_0a9z8W6Lpr9EnFhroKU2ua4Df_PihBA/getMe"
```

## 🚨 Возможные причины ошибки 401

1. **Бот был удален** - создайте нового
2. **Токен был изменен** - получите новый через @BotFather
3. **Бот заблокирован** - создайте нового с другим именем
4. **Опечатка в токене** - проверьте правильность копирования
