# 🔧 Отчет об исправлении проблемы с анализом запросов

## 🎯 Проблема

**Исходная проблема:** Пользователь пишет "Нужна энергия", а бот отвечает рекомендацией для спокойствия (Лаванда) вместо энергии.

**Причина:** Неправильная логика анализа запросов в SmartRouter и handler.js.

## 🔍 Диагностика

### 1. Анализ проблемы
- SmartRouter правильно определял тип запроса как `keyword_search` с ключевым словом "энергия"
- Но в handler.js была дублирующая логика, которая переопределяла результат SmartRouter
- Проблема была в передаче параметров между компонентами

### 2. Найденные проблемы
1. **Дублирующая логика в handler.js** - сначала искал масла, потом использовал SmartRouter
2. **Неправильная передача параметров** - `routeMessage` получал неправильный формат данных
3. **Отсутствующие методы в адаптерах** - AIAdapter не имел нужных методов

## ✅ Исправления

### 1. Исправлена логика в SmartRouter.js
```javascript
// Изменен порядок проверок - сначала ключевые слова, потом поиск масел
// 6. Ключевые слова для общих рекомендаций (ПЕРЕД поиском масел!)
const keywords = this.extractKeywords(normalizedText);
if (keywords.length > 0) {
  return {
    requestType: 'keyword_search',
    chatId: chatId,
    keywords: keywords,
    userQuery: rawText
  };
}
```

### 2. Улучшен метод extractKeywords
```javascript
extractKeywords(normalizedText) {
  const keywordMap = {
    'энергия': ['энергия', 'енергия', 'бодрость', 'активность', 'сил', 'нужна энергия', 'нужна енергия', 'хочу энергии', 'для энергии'],
    'спокойствие': ['спокойствие', 'расслабл', 'релакс', 'покой', 'успокоить', 'хочу расслабиться'],
    // ... другие ключевые слова
  };
}
```

### 3. Убрана дублирующая логика в handler.js
- Удалена логика прямого поиска масел
- Оставлена только SmartRouter логика

### 4. Исправлена передача параметров
```javascript
// Было:
const routingResult = await smartRouter.routeMessage(msg);

// Стало:
const routingResult = await smartRouter.routeMessage({ message: msg });
```

### 5. Добавлены недостающие методы в адаптеры
```javascript
// AIAdapter.js
async getBasicRecommendation(params) {
  return await this.svc.getBasicRecommendation(params);
}

async getMedicalRecommendation(params) {
  return await this.svc.getMedicalRecommendation(params);
}

// OilSearchAdapter.js
async searchDirectOil(params) {
  return await this.svc.searchDirectOil(params);
}

async handleAmbiguousOil(params) {
  return await this.svc.handleAmbiguousOil(params);
}
```
