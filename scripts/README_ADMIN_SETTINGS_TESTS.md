# Тесты админских настроек - Быстрый старт

## Быстрый запуск

```bash
# Комплексный тест (рекомендуется)
npm run test:admin-settings:comprehensive

# Основной тест
npm run test:admin-settings

# Тест интеграции
npm run test:admin-settings:integration

# Тест миграций
npm run test:admin-settings:migrations
```

## Что тестируется

### ✅ Основные функции
- Инициализация админских настроек
- Получение и обновление настроек
- Ежедневная статистика
- Специализированные методы

### ✅ Интеграция
- Админ-панель
- Callback обработчики
- Переключатели настроек
- Другие админские функции

### ✅ Миграции
- Файлы миграций
- SQL синтаксис
- Структура базы данных

### ✅ Обработка ошибок
- Неизвестные callback
- Некорректные параметры
- Graceful degradation

## Режимы работы

### Mock режим (по умолчанию)
- Работает без настройки Supabase
- Использует mock данные
- Идеально для разработки

### Реальный режим
- Требует SUPABASE_URL и SUPABASE_ANON_KEY
- Тестирует реальную базу данных
- Для продакшена

## Устранение неполадок

### Ошибки подключения
```bash
# Проверьте переменные окружения
npm run check:env
```

### Ошибки миграций
```bash
# Запустите миграции
npm run migrate
```

### Проблемы с тестами
```bash
# Запустите тест с подробным выводом
node scripts/testAdminSettingsComprehensive.js
```

## Интеграция с CI/CD

```yaml
# Пример для GitHub Actions
- name: Test Admin Settings
  run: npm run test:admin-settings:comprehensive
```

## Поддержка

Все тесты работают автоматически и не требуют дополнительной настройки. Для полного тестирования настройте Supabase переменные окружения.
