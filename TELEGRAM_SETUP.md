## 🚀 Как запустить бота

### 1. Локальный запуск (для разработки)

```bash
# Запуск сервера
npm start

# Включение webhook
ENABLE_WEBHOOK=true npm start

# Тестирование функций
node scripts/testTelegramBot.js
```

### 2. Настройка для продакшена

#### Шаг 1: Создание бота
1. Откройте Telegram
2. Найдите @BotFather
3. Отправьте команду `/newbot`
4. Следуйте инструкциям для создания бота
5. Сохраните полученный токен

#### Шаг 2: Настройка переменных окружения

Создайте файл `.env` или настройте переменные в Railway:

```env
# Telegram Bot
TELEGRAM_BOT_TOKEN=your_bot_token_here

# Webhook (для Railway)
ENABLE_WEBHOOK=true
WEBHOOK_URL=your-app-url.railway.app

# AI Services (опционально)
OPENAI_API_KEY=your_openai_key
PERPLEXITY_API_KEY=your_perplexity_key

# Database (опционально)
SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_supabase_key

# Admin (опционально)
ADMIN_CHAT_IDS=123456789,987654321
```

#### Шаг 3: Настройка webhook

```bash
# Установка webhook
node scripts/setWebhook.js set

# Проверка статуса
node scripts/setWebhook.js info

# Удаление webhook (если нужно)
node scripts/setWebhook.js delete
```

## 🚨 Устранение неполадок

### Проблема: Бот не отвечает
```bash
# Проверьте логи
tail -f logs/combined.log

# Проверьте статус webhook
node scripts/setWebhook.js info

# Перезапустите сервер
pkill -f "node.*app.js"
npm start
```

### Проблема: Ошибки в логах
- Проверьте переменные окружения
- Убедитесь, что токен бота правильный
- Проверьте доступность внешних API

### Проблема: Webhook не работает
```bash
# Удалите webhook
node scripts/setWebhook.js delete

# Установите заново
node scripts/setWebhook.js set
```
